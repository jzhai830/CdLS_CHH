# CdLS_CHH
This repository contains the single-cell RNA sequencing analysis code associated with the manuscript:

**Cohesin dysfunction causes hypogonadotropic hypogonadism in Cornelia de Lange syndrome**

## Requirements
The analysis was performed in R using the Seurat framework.

## Data availability
The sequencing results and input files used in this analysis are available in the Zenodo repository associated with this manuscript.
